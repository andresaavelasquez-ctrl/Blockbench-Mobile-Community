# Actualización v0.2.0 — Pantalla inmersiva y plugins persistentes

## Cambios

- Oculta la barra de estado y la barra de navegación mediante modo inmersivo.
- Reaplica el modo inmersivo al regresar desde el selector de archivos, diálogos o multitarea.
- Dibuja la WebView de borde a borde y dentro de la zona del recorte/cámara.
- Elimina el padding CSS de `safe-area` para aprovechar todo el panel.
- Copia plugins `.js` instalados desde archivos externos al directorio persistente de la aplicación.
- Restaura esos plugins automáticamente en siguientes inicios.
- Borra la copia persistente cuando el plugin se desinstala.
- Incluye firma estable opcional mediante GitHub Secrets.

## Persistencia

Los plugins externos sobreviven a:

- cerrar y abrir la aplicación;
- cambiar de orientación;
- abrir el selector de archivos;
- reinicios de la actividad Android;
- actualizaciones instaladas con la misma firma.

Se eliminan al:

- desinstalar la aplicación;
- borrar sus datos desde Ajustes;
- desinstalar el plugin desde Blockbench.

## Compatibilidad

El sistema conserva las reglas de Blockbench: un plugin debe ser compatible con la variante web o `both`.
Los plugins exclusivos de Electron/escritorio pueden seguir sin funcionar porque dependen de APIs de Node/Electron.
